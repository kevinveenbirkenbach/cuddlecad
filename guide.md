# 🦄 Das Große Abenteuer des Mojebu – oder: Wie ein mutiger Schreiner CuddleCAD zum Leben erweckte

*Eine Gutenacht-Geschichte für tapfere Windows-Nutzer, erzählt vom weisen Einhorn Glimmerhorn*

---

Es war einmal, in einem fernen Land namens **Windows**, ein Schreiner namens **Mojebu**.

Mojebu war ein guter Mensch. Er konnte wunderbare Dinge aus Holz bauen – Tische, Stühle, Regale. Aber Computer? Computer waren für Mojebu wie ein tiefer, dunkler Wald voller unbekannter Geräusche.

Eines Abends erschien ihm im Traum ein leuchtendes **Einhorn** namens **Glimmerhorn**. Das Einhorn hatte ein goldenes Horn, das sanft summte, und sagte mit warmer Stimme:

> *„Mojebu! Hab keine Angst! Ich führe dich Schritt für Schritt durch den Zauberwald. Folge meinen Hufen – und du wirst CuddleCAD auf deinem Windows-PC zum Leben erwecken!"*

Mojebu nickte tapfer. Das Abenteuer begann.

---

## 🐾 Kapitel 1: Die Werkzeuge des Waldes – Was du brauchst

Glimmerhorn galoppierte zu einem großen Baum und scharrte mit dem Huf:

> *„Bevor wir loslegen, brauchen wir drei magische Werkzeuge. Drei. Nicht vier. Nicht zwei. DREI."*

Ein kleines **Eichhörnchen** namens Zippy hüpfte heran und zählte an seinen winzigen Pfoten ab:

**Werkzeug 1: Python 🐍**

Python ist keine echte Schlange – keine Sorge! Es ist ein Programm, das CuddleCAD versteht.

1. Öffne deinen Browser (das bunte Symbol auf deinem Desktop – Internet Explorer, Chrome, Firefox, egal)
2. Geh zu: **https://www.python.org/downloads/**
3. Klicke auf den großen gelben Knopf **„Download Python 3.x.x"**
4. Lade die Datei herunter und doppelklicke darauf
5. **WICHTIG – SEHR WICHTIG – MEGA WICHTIG:** Bevor du auf „Install Now" klickst, siehst du unten ein kleines Kästchen mit dem Text **„Add Python to PATH"**. Setze dort einen **Haken** ✅. Wenn du das nicht tust, weint das Einhorn.
6. Klicke auf „Install Now"
7. Warte. Trinke einen Schluck Wasser.
8. Klicke auf „Close"

*Ein kleiner **Schmetterling** flatterte vorbei und flüsterte: „Python ist jetzt dein Freund!"*

---

**Werkzeug 2: Git 🦊**

Git ist ein roter Fuchs, der Dateien aus dem Internet holen kann.

1. Geh zu: **https://git-scm.com/download/win**
2. Die Datei lädt automatisch herunter
3. Doppelklicke auf die Datei
4. Klicke immer auf **„Next"** – gefühlt 47 Mal. Ändere NICHTS. Einfach Next, Next, Next, Next...
5. Am Ende: **„Install"** klicken, dann **„Finish"**

*Das Eichhörnchen Zippy nickte zufrieden: „Gut gemacht, Mojebu!"*

---

**Werkzeug 3: Visual Studio Code (VS Code) 🦁**

Das ist dein Zuhause für den Code. Ein gemütliches Löwengehege, in dem dein Code sicher schläft.

1. Geh zu: **https://code.visualstudio.com/**
2. Klicke auf den blauen Knopf **„Download for Windows"**
3. Doppelklicke auf die heruntergeladene Datei
4. Klicke immer auf **„Weiter"** / **„Next"** / **„Ich stimme zu"**
5. Bei der Frage nach den „Additional Tasks": setze **alle** Häkchen ✅✅✅✅
6. Klicke auf „Installieren" → „Fertigstellen"

---

## 🌟 Kapitel 2: Das Zauberbuch aus dem Netz holen

Glimmerhorn galoppierte zu einem glitzernden See. Am Ufer spiegelte sich der Code von CuddleCAD.

> *„Jetzt holen wir das Zauberbuch – den Code – auf deinen Computer!"*

Ein **Otter** namens Plop tauchte auf und zeigte den Weg:

### So öffnest du das schwarze Fenster (Kommandozeile):

1. Drücke gleichzeitig die Tasten **Windows-Taste** (die mit dem Windows-Symbol) und **R**
2. Es erscheint ein kleines Fenster
3. Tippe **`cmd`** ein und drücke **Enter**
4. Ein schwarzes Fenster erscheint – das ist die **Kommandozeile**. Sie sieht beängstigend aus. Sie beißt nicht.

### Jetzt tippe diese magischen Worte ein (abschreiben!):

```
git clone https://github.com/kevinveenbirkenbach/cuddlecad.git
```

Drücke **Enter**.

Du siehst viele Zeilen Text. Das ist normal. Der Otter Plop holt gerade alle Dateien.

Danach tippe:

```
cd cuddlecad
```

Drücke **Enter**.

Du bist jetzt im Zuhause von CuddleCAD. Herzlichen Glückwunsch! 🎉

*Eine **Schildkröte** namens Langsam applaudierte gemächlich.*

---

## 🚀 Kapitel 3: CuddleCAD zum Leben erwecken

Das Einhorn Glimmerhorn senkte sein Horn auf das schwarze Fenster und flüsterte:

> *„Sprich die Beschwörungsformel!"*

Tippe in das schwarze Fenster:

```
python app.py
```

Drücke **Enter**.

Du siehst so etwas:

```
CuddleCAD laeuft auf http://0.0.0.0:8000
```

**Das bedeutet: CuddleCAD lebt!** 🎊

Jetzt öffne deinen Browser und tippe in die Adressleiste ganz oben:

```
http://localhost:8000
```

Drücke **Enter**.

**Und da ist es.** CuddleCAD. Dein CuddleCAD. Auf deinem Computer. Lokal. Ganz allein für dich.

*Ein **Reh** sprang aus dem Wald und machte einen Luftsprung vor Freude.*

---

## 🛑 Kapitel 4: CuddleCAD wieder stoppen

Wenn du fertig bist und das schwarze Fenster schließen möchtest:

Klicke in das schwarze Fenster und drücke gleichzeitig **Strg** + **C**.

Das Programm hört auf. Das schwarze Fenster kann jetzt geschlossen werden.

*„Gute Nacht, CuddleCAD"*, sagte Mojebu.*

---

## 🔧 Kapitel 5: CuddleCAD in VS Code öffnen und anschauen

Das Einhorn erklärte sanft:

> *„Jetzt zeige ich dir, wo der Code wohnt. Dort kannst du der Künstlichen Intelligenz sagen, was sie ändern soll."*

1. Öffne **VS Code** (das blaue Viereck auf deinem Desktop)
2. Klicke oben links auf **„File"** → **„Open Folder"**
3. Suche den Ordner **„cuddlecad"** – er liegt wahrscheinlich in **„C:\Users\DEIN_NAME\cuddlecad"**
4. Klicke auf den Ordner und dann auf **„Ordner auswählen"**

Jetzt siehst du links eine Liste mit Dateien. Die wichtigsten sind:

| Datei | Was sie tut |
|-------|-------------|
| `app/static/js/app.js` | Das Herzstück – hier passiert alles Magische |
| `app/index.html` | Die Buttons und das Layout |
| `app/static/css/style.css` | Farben, Formen, Schönheit |
| `app.py` | Der Kellner, der alles serviert |
| `requirements.md` | Die Liste aller gewünschten Funktionen |

*Ein **Pinguin** namens Klaus wies mit seiner Flosse auf die Dateien und nickte ernst.*

## 🤖 Kapitel 6: Die Künstliche Intelligenz beauftragen – Der mächtigste Teil!

Glimmerhorn's Horn leuchtete golden auf:

> *„Mojebu! Jetzt kommt der Trick, mit dem du ALLES verändern kannst, ohne selbst zu programmieren. Du brauchst nur... zu SCHREIBEN!"*

Ein **Papagei** namens Codinho setzte sich auf Mojebus Schulter und erklärte:

### Schritt 1: Claude Code installieren

Claude Code ist die KI, die CuddleCAD für dich weiterentwickelt.

1. Öffne das schwarze Fenster (Kommandozeile) erneut (Windows + R → `cmd` → Enter)
2. Tippe:

```
npm install -g @anthropic-ai/claude-code
```

Warte. Das dauert etwas. Ein **Schnabeltier** schläft währenddessen.

Falls du eine Fehlermeldung bekommst weil `npm` nicht gefunden wird: Du brauchst erst **Node.js**! Geh zu **https://nodejs.org** → lade die **LTS-Version** herunter → installiere sie (immer auf „Next" klicken) → dann versuche den `npm`-Befehl erneut.

### Schritt 2: Den Code-Ordner öffnen

Im schwarzen Fenster, tippe:

```
cd cuddlecad
```

(Falls du das schwarze Fenster geschlossen hast: öffne ein neues und tippe erst `cd cuddlecad`)

### Schritt 3: Claude Code starten

```
claude
```

Drücke Enter. Folge den Anweisungen auf dem Bildschirm (Email eingeben, bestätigen).

### Schritt 4: Der Schreiner spricht zur KI

Jetzt kommt die Magie! Du schreibst der KI einfach auf Deutsch, was du willst. Zum Beispiel:

---

**Beispiel 1 – Neue Funktion:**
```
Füge einen neuen Button "Tisch" hinzu, der automatisch vier Beine und eine Platte als Bretter erstellt.
```

**Beispiel 2 – Farbe ändern:**
```
Ändere die Hintergrundfarbe des Canvas von hellgelb zu einem warmen Holzbraun.
```

**Beispiel 3 – Fehler beheben:**
```
Wenn ich einen Eckpunkt ziehe, verschwindet manchmal das Brett. Bitte fix das.
```

**Beispiel 4 – Neue Bearbeitung:**
```
Füge eine neue Bearbeitung "Zapfen" hinzu, die ein rechteckiges Stück aus der Stirnseite des Bretts herausschneidet.
```

---

*Der Papagei Codinho rief aufgeregt: „Siehst du, Mojebu? Du bist jetzt ein Software-Architekt! Du entscheidest, was gebaut wird – die KI baut es!"*

Das Einhorn Glimmerhorn nickte weise:

> *„Denk daran: Je genauer du beschreibst, was du willst, desto besser wird das Ergebnis. Sag der KI: WAS soll es tun? WIE soll es aussehen? WARUM brauchst du das?"*

## 💾 Kapitel 7: Deine Änderungen speichern

Ein **Biber** namens Holger erklärte die Kunst des Speicherns:

Wenn die KI Änderungen gemacht hat und alles funktioniert, speichere deine Arbeit mit:

Im schwarzen Fenster (im `cuddlecad`-Ordner):

```
git add .
git commit -m "Meine tolle neue Funktion"
```

Das ist wie ein Foto von deinem Fortschritt. Du kannst jederzeit zu diesem Foto zurückkehren.

---

## 🐛 Kapitel 8: Wenn etwas schiefgeht – Der Drachen-Bezwinger

Ein kleiner **Drache** namens Fehlermeldung blockierte manchmal den Weg. Aber Mojebu lernte, ihn zu zähmen:

### Problem: "python wird nicht erkannt"
**Lösung:** Python wurde ohne PATH installiert. Deinstalliere Python (Systemsteuerung → Programme) und installiere es erneut – diesmal MIT dem Häkchen bei „Add Python to PATH"!

### Problem: Die Seite lädt nicht (localhost:8000 funktioniert nicht)
**Lösung:** Läuft `python app.py` noch im schwarzen Fenster? Das schwarze Fenster darf NICHT geschlossen sein, solange du CuddleCAD nutzt!

### Problem: "git wird nicht erkannt"
**Lösung:** Starte deinen Computer einmal neu und versuche es erneut.

### Problem: Die KI macht etwas falsch
**Lösung:** Schreibe der KI: *„Das war falsch, weil [erkläre das Problem]. Mache das rückgängig und versuche es anders."*

*Der kleine Drache Fehlermeldung wurde damit immer kleiner und kleiner, bis er harmlos war wie ein Gecko.*
